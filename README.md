# Bakery Shop Homepage

A simple and responsive bakery shop homepage built using **HTML** and **CSS**. This project showcases a clean and modern landing page design for a bakery website.

## Features

- Responsive homepage layout
- Modern and clean design
- Navigation bar
- Hero section
- Product/Services section
- Contact section
- Pure HTML and CSS (no JavaScript)

## Technologies Used

- HTML5
- CSS3

## Future Improvements

- Add JavaScript for interactive features.
- Improve mobile responsiveness.
- Add animations and transitions.
- Include an online ordering system.
